Nhập câu hỏi từ Excel

Hướng dẫn nhập file Excel/CSV:
  • File phải có cấu trúc gồm 7 cột. Dòng đầu tiên là tiêu đề (sẽ bị bỏ qua).
  • Cột 1: Nội dung câu hỏi
  • Cột 2, 3, 4, 5: Đáp án A, B, C, D
  • Cột 6: Đáp án đúng (Chỉ nhập số từ 1 đến 4, tương ứng A,B,C,D)
  • Cột 7: Giải thích (Tùy chọn)
